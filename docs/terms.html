Description of Service

contacts2notion is an open-source tool that syncs contact information between Google Contacts and Notion databases. The software:

  • Provides 2-way synchronization between Google Contacts and Notion
  • Uses OAuth 2.0 to access your Google Contacts data
  • Uses the Notion API to read and write to your Notion database
  • Can be run locally or via GitHub Actions
